# 🚀 Portafolio Personal

Un portafolio moderno y responsive desarrollado con React, TypeScript, Vite y Tailwind CSS.

## ✨ Características

- 🎨 **Diseño Moderno**: Interfaz limpia y profesional con animaciones suaves
- 🌙 **Modo Oscuro**: Soporte completo para tema claro y oscuro
- 📱 **Responsive**: Optimizado para todos los dispositivos
- ⚡ **Rendimiento**: Construido con Vite para máxima velocidad
- 🎯 **SEO Optimizado**: Meta tags y estructura semántica
- 🎭 **Animaciones**: Transiciones fluidas con Framer Motion

## 🛠️ Tecnologías Utilizadas

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Animaciones**: Framer Motion
- **Iconos**: Lucide React
- **Enrutamiento**: React Router DOM
- **Build Tool**: Vite
- **Linting**: ESLint

## 🚀 Instalación

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/tu-usuario/portafolio-personal.git
   cd portafolio-personal
   ```

2. **Instala las dependencias**
   ```bash
   npm install
   ```

3. **Inicia el servidor de desarrollo**
   ```bash
   npm run dev
   ```

4. **Abre tu navegador**
   ```
   http://localhost:5173
   ```

## 🎨 Personalización

### Colores
Los colores principales se pueden personalizar en `tailwind.config.js`:

```javascript
colors: {
  primary: {
    50: '#eff6ff',
    // ... más tonos
    900: '#1e3a8a',
  }
}
```

### Contenido
- Actualiza la información personal en los componentes
- Modifica los enlaces de redes sociales
- Personaliza las secciones según tus necesidades

## 📝 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye para producción
- `npm run preview` - Vista previa de la build
- `npm run lint` - Ejecuta el linter
- `npm run type-check` - Verifica tipos de TypeScript
